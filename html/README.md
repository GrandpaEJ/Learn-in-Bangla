HTML-এর সকল (বা প্রায় সব) স্ট্যান্ডার্ড ট্যাগের একটি পূর্ণাঙ্গ তালিকা নিচে ক্যাটাগরি অনুযায়ী দেওয়া হলো। এটি আপনার জন্য রেফারেন্স হিসেবে কাজ করবে।

### ১. মূল কাঠামো এবং মেটাডাটা (Basic & Metadata)

এই ট্যাগগুলো একটি ওয়েবসাইটের ভিত্তি এবং ব্রাউজারের জন্য প্রয়োজনীয় তথ্য বহন করে।

| ট্যাগ | বিবরণ (Description) |
| --- | --- |
| `<!DOCTYPE>` | এটি HTML সংস্করণ (যেমন HTML5) নির্দেশ করে। |
| `<html>` | HTML ডকুমেন্টের রুট বা প্রধান কন্টেইনার। |
| `<head>` | মেটা-তথ্য, টাইটেল এবং স্ক্রিপ্ট ধারণ করে। |
| `<title>` | ব্রাউজার ট্যাবে ওয়েবসাইটের নাম দেখায়। |
| `<meta>` | ক্যারেক্টার সেট, কি-ওয়ার্ড এবং অথর সম্পর্কিত তথ্য। |
| `<link>` | এক্সটার্নাল ফাইল (যেমন CSS) যুক্ত করার জন্য। |
| `<style>` | ইন্টারনাল CSS লেখার জন্য। |
| `<script>` | জাভাস্ক্রিপ্ট কোড বা ফাইল যুক্ত করার জন্য। |
| `<base>` | ডকুমেন্টের সকল রিলেটিভ লিঙ্কের জন্য বেস URL সেট করে। |

### ২. কন্টেন্ট সেকশন এবং লেআউট (Sectioning & Layout)

ওয়েবসাইটের গঠন বা লেআউট তৈরির জন্য এই ট্যাগগুলো ব্যবহৃত হয়।

| ট্যাগ | বিবরণ (Description) |
| --- | --- |
| `<body>` | পেজের দৃশ্যমান সকল কন্টেন্ট এখানে থাকে। |
| `<header>` | পেজ বা সেকশনের উপরের অংশ (লোগো, মেনু)। |
| `<nav>` | নেভিগেশন লিঙ্কগুলোর সেট। |
| `<main>` | পেজের প্রধান ইউনিক কন্টেন্ট। |
| `<section>` | ডকুমেন্টের একটি নির্দিষ্ট বিভাগ। |
| `<article>` | একটি স্বতন্ত্র কন্টেন্ট (যেমন ব্লগ পোস্ট বা নিউজ)। |
| `<aside>` | মূল কন্টেন্টের পাশে থাকা তথ্য (Sidebar)। |
| `<footer>` | পেজ বা সেকশনের নিচের অংশ (কপিরাইট তথ্য)। |
| `<div>` | কন্টেন্ট গ্রুপ করার জন্য জেনেরিক কন্টেইনার। |
| `<h1> - <h6>` | ছয়টি স্তরের হেডিং বা শিরোনাম। |

### ৩. টেক্সট ফরম্যাটিং (Text Semantics)

লেখাকে সুন্দর এবং অর্থবহ করার জন্য এই ট্যাগগুলো ব্যবহার করা হয়।

| ট্যাগ | বিবরণ (Description) |
| --- | --- |
| `<p>` | প্যারাগ্রাফ বা অনুচ্ছেদ। |
| `<a>` | হাইপারলিঙ্ক (এক পেজ থেকে অন্য পেজে যাওয়া)। |
| `<span>` | ইনলাইন কন্টেন্টকে স্টাইল করার জন্য ব্যবহৃত হয়। |
| `<b>` / `<strong>` | লেখাকে মোটা (Bold) বা গুরুত্বপূর্ণ বোঝাতে। |
| `<i>` / `<em>` | লেখাকে বাঁকা (Italic) বা জোর দিতে। |
| `<u>` / `<ins>` | লেখার নিচে আন্ডারলাইন দিতে। |
| `<s>` / `<del>` | লেখার মাঝখানে দাগ (Strikethrough) দিয়ে কাটা বোঝাতে। |
| `<sub>` | সাবস্ক্রিপ্ট (যেমন )। |
| `<sup>` | সুপারস্ক্রিপ্ট (যেমন )। |
| `<mark>` | হাইলাইট করা টেক্সট। |
| `<code>` | কম্পিউটার কোড দেখানোর জন্য। |
| `<pre>` | প্রি-ফরম্যাটেড টেক্সট (যেমন আছে তেমনই দেখাবে)। |
| `<blockquote` | অন্য কোনো উৎস থেকে নেওয়া বড় উক্তি। |
| `<abbr>` | সংক্ষিপ্ত রূপ বা অ্যাব্রিভিয়েশন (যেমন HTML)। |

### ৪. লিস্ট এবং টেবিল (Lists & Tables)

তথ্য সাজিয়ে দেখানোর জন্য এই ট্যাগগুলো অপরিহার্য।

| ট্যাগ | বিবরণ (Description) |
| --- | --- |
| `<ul>` | আন-অর্ডারড বা বুলেট পয়েন্ট লিস্ট। |
| `<ol>` | অর্ডারড বা নাম্বারিং করা লিস্ট। |
| `<li>` | লিস্টের আইটেম। |
| `<dl>` | ডেসক্রিপশন লিস্ট। |
| `<dt>` | ডেসক্রিপশন টার্ম বা নাম। |
| `<dd>` | টার্মের বিস্তারিত বিবরণ। |
| `<table>` | টেবিল তৈরি করে। |
| `<tr>` | টেবিলের রো (Row)। |
| `<th>` | টেবিলের হেডার সেল। |
| `<td>` | টেবিলের ডাটা সেল। |
| `<thead>` | টেবিলের উপরের অংশ। |
| `<tbody>` | টেবিলের মূল কন্টেন্ট অংশ। |
| `<tfoot>` | টেবিলের নিচের অংশ। |

### ৫. ফর্ম এবং ইউজার ইনপুট (Forms & Inputs)

ইউজারের কাছ থেকে তথ্য নেওয়ার জন্য এই ট্যাগগুলো ব্যবহৃত হয়।

| ট্যাগ | বিবরণ (Description) |
| --- | --- |
| `<form>` | ইউজার ইনপুটের জন্য ফর্ম কন্টেইনার। |
| `<input>` | বিভিন্ন ধরণের ইনপুট ফিল্ড (Text, Radio, Checkbox)। |
| `<label>` | ইনপুট ফিল্ডের নাম। |
| `<button>` | ক্লিকযোগ্য বাটন। |
| `<textarea>` | অনেকগুলো লাইন লেখার বক্স। |
| `<select>` | ড্রপ-ডাউন লিস্ট। |
| `<option>` | ড্রপ-ডাউন লিস্টের আইটেম। |
| `<optgroup>` | অপশনগুলোকে গ্রুপ করার জন্য। |
| `<fieldset>` | ফর্মের সম্পর্কিত তথ্য গ্রুপ করতে। |
| `<legend>` | fieldset-এর ক্যাপশন বা টাইটেল। |

### ৬. মিডিয়া এবং এমবেড (Media & Embedded)

ছবি, অডিও এবং ভিডিও যুক্ত করার জন্য।

| ট্যাগ | বিবরণ (Description) |
| --- | --- |
| `<img>` | ছবি প্রদর্শনের জন্য। |
| `<audio>` | অডিও ফাইল প্লে করার জন্য। |
| `<video>` | ভিডিও ফাইল প্লে করার জন্য। |
| `<source>` | মিডিয়া ফাইলের উৎস নির্ধারণ করে। |
| `<iframe>` | অন্য কোনো ওয়েবসাইট বা পেজ এমবেড করতে। |
| `<canvas>` | গ্রাফিক্স বা ড্রয়িং করার জন্য (JS প্রয়োজন)। |
| `<svg>` | ভেক্টর গ্রাফিক্স যুক্ত করতে। |

### ৭. অন্যান্য গুরুত্বপূর্ণ ট্যাগ (Miscellaneous)

| ট্যাগ | বিবরণ (Description) |

### অবশিষ্ট ও বিশেষায়িত HTML ট্যাগসমূহ

| ট্যাগ | বিবরণ (Description) |
| --- | --- |
| `<address>` | কোনো ব্যক্তি বা প্রতিষ্ঠানের যোগাযোগের তথ্য প্রদানের জন্য। |
| `<abbr>` | সংক্ষিপ্ত শব্দের পূর্ণরূপ বোঝাতে (Abbreviation)। |
| `<bdi>` | টেক্সটকে তার চারপাশের টেক্সট থেকে আলাদা করতে (Bi-directional Isolation)। |
| `<bdo>` | টেক্সটের দিক পরিবর্তন করতে (Bi-directional Override)। |
| `<cite>` | কোনো কাজের শিরোনাম (যেমন: বই বা সিনেমার নাম) নির্দেশ করতে। |
| `<kbd>` | কীবোর্ড ইনপুট বা শর্টকাট দেখানোর জন্য (যেমন: Ctrl+C)। |
| `<samp>` | কম্পিউটার প্রোগ্রামের স্যাম্পল আউটপুট দেখাতে। |
| `<var>` | গাণিতিক বা প্রোগ্রামিং ভ্যারিয়েবল (Variable) বোঝাতে। |
| `<wbr>` | ব্রাউজারকে বলা যে এখানে লাইন ব্রেক করা সম্ভব (Word Break Opportunity)। |
| `<ruby>` | পূর্ব এশীয় ভাষার উচ্চারণ দেখানোর জন্য (Ruby Annotation)। |
| `<rt>` | রুবি অ্যানোটেশনের টেক্সট বা উচ্চারণ। |
| `<rp>` | রুবি অ্যানোটেশন সাপোর্ট না করা ব্রাউজারের জন্য ব্র্যাকেট। |
| `<data>` | কন্টেন্টকে মেশিন-রিডেবল ডেটার সাথে লিঙ্ক করতে। |
| `<time>` | তারিখ এবং সময়কে মেশিন-রিডেবল ফরমেটে প্রকাশ করতে। |
| `<meter>` | একটি নির্দিষ্ট সীমার মধ্যে পরিমাপ দেখাতে (যেমন: ডিস্ক স্পেস)। |
| `<progress>` | কোনো কাজের অগ্রগতি (Task progress) দেখাতে। |
| `<output>` | কোনো ক্যালকুলেশনের ফলাফল দেখাতে। |
| `<map>` | ইমেজ ম্যাপ (ছবির নির্দিষ্ট অংশে ক্লিক করার সুবিধা) তৈরির জন্য। |
| `<area>` | ইমেজ ম্যাপের ক্লিকযোগ্য এরিয়া নির্ধারণ করতে। |
| `<canvas>` | জাভাস্ক্রিপ্টের মাধ্যমে গ্রাফিক্স বা অ্যানিমেশন তৈরি করতে। |
| `<noscript>` | যদি ব্রাউজারে জাভাস্ক্রিপ্ট বন্ধ থাকে তবে কী দেখাবে তা নির্ধারণ করতে। |
| `<object>` | এক্সটার্নাল রিসোর্স (যেমন: PDF বা ফ্ল্যাশ) এমবেড করতে। |
| `<param>` | `<object>` ট্যাগের প্যারামিটার পাস করতে। |
| `<embed>` | এক্সটার্নাল কন্টেন্ট বা প্লাগইন যুক্ত করতে। |
| `<picture>` | বিভিন্ন ডিভাইসের জন্য ছবির ভিন্ন ভিন্ন ভার্সন দিতে। |
| `<track>` | ভিডিও বা অডিওর জন্য টেক্সট ট্র্যাক (Subtitles) যোগ করতে। |

### কিছু বাতিল (Deprecated) ট্যাগ:

আগে `<center>`, `<font>`, `<strike>`, `<big>`, `<marquee>` ট্যাগগুলো ব্যবহৃত হতো, কিন্তু আধুনিক HTML5-এ এগুলো আর ব্যবহার করা হয় না। এগুলোর কাজ এখন **CSS** দিয়ে করা হয়।
